Promote Windows NT, 200x, XP, Vista and Seven user to Administrator - Rescapp - Documentation
  • Step 1

    Rescapp searches for partitions, on your drive with system32/config/sam files on them.
  • Step 2

    If such partitions have been found, Rescapp displays a list of the partitions. At the same time you are prompted:

    Which partition?

    You are supposed to choose, your Windows partition, (you will probably only see one), and press the OK button.
  • Step 3

    Rescapp searches for System users based on previous information.
  • Step 4

    You are prompted:

    Choose Windows user to promote to Admin

    You are supposed to select, the user which you want to, promote to Administrator.
  • Step 5

    You are informed about the user promotion process.

    You either get:

    The user was promoted to Admin OK! :) or

    or

    The user was not promoted to Admin. Something went wrong! :(
  • Step 6

    A log file called winpromote.log.txt can be found in the Desktop logs folder.
