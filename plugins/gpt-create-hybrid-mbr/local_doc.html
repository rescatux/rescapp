Create a hybrid MBR on a GPT partition - Rescapp - Documentation
  • Step 1

    Rescapp searches for hard drives on your computer.
  • Step 2

    If such hard drives have been found, Rescapp displays a list of the hard drives. At the same time you are prompted:

    Which hard drive?

    You are supposed to choose, the hard drive which has a GPT partition, on which you want to have a hybrid MBR installed.
  • Step 3

    Rescapp with all the gathered information, is going to create a hybrid MBR on your GPT hard drive.Please wait. Currently with (Rescatux 0.40b1), this step is time consuming but no Please wait message has been implemented.
  • Step 4

    You are informed about the hybrid mbr creation process.

    You either get:

    Success! The hybrid MBR was created OK! :) or

    or

    Failure! The hybrid MBR was not created. Something went wrong! :(
  • Step 5

    A log file called gpt-create-hybrid-mbr_log.txt can be found in the Desktop logs folder.
