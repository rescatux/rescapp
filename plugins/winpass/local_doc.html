Reset Windows NT, 200x, XP, Vista and Seven password - Rescapp - Documentation
  • Step 1

    Rescapp searches for partitions on your drive with system32/config/sam files.
  • Step 2

    If such partitions have been found, Rescapp displays a list of the partitions. At the same time you are prompted:

    Which partition?

    You are supposed to choose your Windows (you will probably only see one) and press the OK button.

  • Step 3

    Rescapp searches for System users based on previous information.
  • Step 4

    You are prompted:

    Choose Windows user to reset its password

    You are supposed to select the user which password you want to reset.

  • Step 5

    You are informed about the windows password reset process.

    You either get:

    Windows password was reset OK! :)

    or

    Failure! Windows password was not reset. Something went wrong! :(

  • Step 6

    A log file called winpass.log.txt can be found in the Desktop logs folder.
